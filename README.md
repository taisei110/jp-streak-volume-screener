# 日本株 連騰・出来高スクリーナー

## 📈 概要
直近数日間で**「連続して株価が上昇（陽線）」**しており、かつ**「出来高が急増」**している日本株を自動で抽出するスクリーニングツールです。
機関投資家の買い集めや、強いトレンドの初動に乗るための候補銘柄を効率的に見つけることを目的としています。

---

## ✨ 主な特徴

1. **正確な出来高・異常値検知**
   - 連騰が始まる「前」の10日間の中央値を基準とするため、連騰中の出来高増に引っ張られず、正確な「平常時からの変化率（急増度合い）」を計算します。
2. **高速・軽量なデータ処理**
   - `yfinance` を用いて無料で株価を取得。
   - データベースに `DuckDB` を採用し、2回目以降は「差分データのみ」をダウンロードするため、全上場銘柄（約3800銘柄）の処理が非常に高速です。
3. **見やすいモダンなHTMLレポート**
   - スクリーニング結果は、見やすくデザインされた美しいHTMLレポート (`screen_result.html`) として出力されます。
4. **Discordへの自動通知**
   - 抽出完了後、結果のサマリー（トップ銘柄など）をご自身のDiscordに自動で通知します。

---

## 🔍 スクリーニング条件（デフォルト設定）

スクリプト内の設定値（定数）を変更することで、条件を自由にカスタマイズ可能です。

* **連騰日数 (`STREAK_DAYS`)**: 3日
  * 3日連続で「前日比プラス」かつ「陽線（終値 > 始値）」であること。
* **出来高の基準期間 (`BASE_WINDOW`)**: 10日
  * 連騰が始まる直前10日間の出来高の中央値を「平常時の出来高」と定義します。
* **出来高の判定閾値 (`VOL_MULT`)**: 1.3倍
  * 平常時の出来高に対して、連騰中の出来高が1.3倍以上に急増している銘柄を抽出します。
* **出来高判定モード (`VOL_MODE`)**: "all"
  * `all`: 連騰している全ての日において、出来高が閾値を超えている（厳しめ）
  * `avg`: 連騰中の出来高の「平均」が、閾値を超えている（緩め）

## 🚀 使い方

### 1. 必要なパッケージのインストール
初回のみ、以下のコマンドで必要なライブラリをインストールしてください。
```bash
pip install yfinance duckdb pandas openpyxl xlrd schedule requests
```

### 2. スクリプトの実行方法

* **通常実行（最新データを取得して抽出）**
  ```bash
  python jp_streak_volume_screener.py
  ```
  ※毎日の自動実行（タスクスケジューラ）にはこのコマンドを登録するのがオススメです。

* **既存データで再抽出（取得スキップ）**
  ```bash
  python jp_streak_volume_screener.py --no-fetch
  ```
  ※条件（日数や倍率）だけを書き換えて、すぐに結果を確認したい時に便利です。

* **常駐モード（黒い画面を開きっぱなしにする）**
  ```bash
  python jp_streak_volume_screener.py --daemon
  ```
  ※平日の20:00に自動で抽出が走ります。

## 🔔 Discordへの自動通知設定

結果をスマホやPCのDiscordで受け取りたい場合は、以下の設定を行ってください。

1. Discordでお好きなサーバーの「チャンネルの編集」＞「連携サービス」＞「ウェブフックを作成」から**ウェブフックURLをコピー**します。
2. `jp_streak_volume_screener.py` を開き、61行目付近にある `DISCORD_WEBHOOK_URL` にURLを貼り付けます。
   ```python
   DISCORD_WEBHOOK_URL = "https://discord.com/api/webhooks/xxxx/yyyy..."
   ```

---

## 📂 出力ファイル

実行が完了すると、同じフォルダに以下のファイルが生成・上書きされます。

* **`screen_result.html`**: 結果が見やすくフォーマットされたレポート画面。お気に入り（ブックマーク）に入れておくと便利です。
* **`screen_result.csv`**: Excelなどで独自に分析したい場合に使用する生データ。
* **`data/prices.duckdb`**: 取得した株価データが蓄積されるデータベースファイル。
